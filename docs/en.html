Download and Usage

  1. 1. Download the file
  2. 2. Configure the server.ini file
    1. This file is located in the launcher directory
    2. Open the server.ini file with a text editor.
    3. If it doesn't exist, the launcher will automatically generate a default configuration file.
    4. Of course, you can actually make most configurations in the launcher interface.
    5. However, we do not provide a download link for the server jar file.
    6. Download file? You can configure where the server jar file should be downloaded in the ini file, and our program will call the Windows API to download the file.
    7. Please ensure your network connection is working properly to download the required files smoothly.
    8. If you encounter problems during the download process, you can try manually downloading the server jar file and placing it in the specified directory.
    9. The launcher will automatically detect the jar file and load it (this feature has not been implemented yet).
    10. Here is an example configuration for the server.ini file:
      11.             [Server]
            JARname=server.jar
            RAMmax=1024
            RAMmin=128
            Downloadurl=https://example.com/downloads/server.jar

        1. Modify [Server]

      1. This item does not need to be modified

        2. 2. Modify JARname

      2. This item represents the name of the server jar file. The launcher will automatically look for the jar file based on this item.

        3. 3. Modify RAMmax and RAMmin

      3. This item represents the maximum and minimum memory available to the server. Adjust as needed.

        4. 4. Modify Downloadurl

      4. This item represents the download link for the server jar file. Modify as needed.
      5. "https://example.com/downloads/server.jar" is a sample link, please modify it according to the actual situation. The downloaded file will be saved in the launcher directory.
      6. The name of the downloaded file must match the JARname item.
